Linked Signal

    

        A linked signal is a writable signal whose value is derived from another signal, but can also be manually overridden.

   



Parent and child linked signal
    

        A linked signal is a writable signal that derives its value from another signal but can be independently updated when needed.
        Behavior:
	        childCount() starts with 10
	        If parentCount changes → childCount updates
	        If you manually set childCount → it diverges
